How It Works

The Ad Arena benchmark evaluates how well LLMs can learn to run search-ad campaigns from data alone. Each model acts as a bidder in a simulated ad auction, setting daily keyword bids, budgets, and ad copy over a 30-day episode.

Evaluation Suites

  • Campaign Efficiency — Total profit (revenue minus spend) across all 30 days.
  • Strategy Optimization — Profit from only the final 7 days (days 24–30).
